## 简单入门
### 1.命令
#### 创建新文章
``` bash
$ hexo new "文章名"  //创建新的博客文章
```

#### 发布文章
``` bash
//当你编辑完新的文章，需要下面的命令更新并发布到github上
$ hexo clean 
$ hexo g -d
```
