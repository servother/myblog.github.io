Linux命令学习之ps

发表于 2018-10-19 | 分类于 Linux

ps命令详解

Linux上进程有五种状态

  1. 运行（正在运行或在运行队列中等待）
  2. 中断（休眠中，受阻，在等待某个条件的行程或接受到信号）
  3. 不可中断
  4. 僵死（进程已经终止，但进程描述符存在）
  5. 停止（进程收到SIGSTOP, SIGSTP, SIGTIN, SIGTOU信号后停止运行）

ps工具标识进程的五种状态码

  1. D：不可中断
  2. R：运行中
  3. S：中断
  4. T：停止
  5. Z：僵死
